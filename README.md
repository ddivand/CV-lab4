# CV-lab4
###### web-app
В папке находится requirements.txt, часть зависимостей там пересекается с теми, что нужны для старта консольного приложения.
На nano ошибка несовместимостимсоти scipy с numpy.
Приложение разворачивается при помощи веб-сервера uvicorn и fastapi.

Для старта приложения:
```
python3 server.py serve
```
Приложение разворачиваетс на **localhost:5555**
